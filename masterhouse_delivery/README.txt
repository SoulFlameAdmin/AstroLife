MASTERHOUSE DELIVERY — локален пакет

КЪДЕ ДА ГО СЛОЖИШ
Разархивирай ZIP-а в:
E:\SF_DOSTAVKI\masterhouse_delivery

КАК СЕ ПУСКА
1) Отвори папката.
2) Стартирай start_masterhouse.bat
3) Ще се отворят:
   - локален сървър на http://127.0.0.1:8123
   - launcher.html
   - отделен прозорец, който изтегля продуктите

ФАЙЛОВЕ
- launcher.html           -> начална страница с бутона/логото
- catalog.html            -> визуален каталог
- update_masterhaus.py    -> scraper за продукти, цени, категории, URL, снимки
- start_masterhouse.bat   -> пуска всичко
- update_masterhaus.bat   -> само обновява данните
- open_only.bat           -> само отваря локалния сайт
- data/products.json      -> всички записани продукти
- data/products.csv       -> CSV export
- data/meta.json          -> статус на scraper-а

ВАЖНО
- Първото пълно обновяване може да отнеме време, защото сайтът има много категории и страници.
- Каталогът се обновява автоматично през няколко секунди, докато scraper-ът записва данните.
- Ако сайтът на MASTERHAUS промени HTML структурата си, scraper-ът може да иска лека корекция.
- В пакета има начални примерни продукти. След обновяване ще бъдат заменени с реално изтеглени данни.

АКО НЕ ТРЪГНЕ
- Инсталирай Python 3.
- Провери дали имаш интернет.
- Стартирай update_masterhaus.bat отделно и виж дали има грешка в прозореца.
